## Текущая проблема ??
**Ввод серий стрельбы:** Access Violation при автоматическом переходе ячеек (Enter)

### Симптомы:
- При ручном клике мышкой ? Enter: работает без ошибок
- При автоматическом переходе MoveNext: Access Violation
- Ошибка в System.Generics.Collections.TDictionary по адресу 002A727B

### Последовательность:
1. Ввод результата серии (например "77")
2. Нажатие Enter
3. OnEditKeyPress ? KillEditor(true) ? MoveNext
4. Crash

### Файлы для исправления:
- `wb_seriesedit.pas` - основной файл редактора серий
- Классы: TShooterSeriesEditor
- Методы: OnEditKeyPress, MoveNext, KillEditor, CreateEditor
