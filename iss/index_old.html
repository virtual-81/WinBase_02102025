WinBASE

Программа была разработана специально для сборной олимпийской команды СССР по пулевой стрельбе для хранения и учёта результатов, для ведения рейтинга спортсменов, а также в дальнейшем для проведения соревнований, формирования стартовых и финальных протоколов.

Это ПО - свободно распространяемый продукт, является интеллектуальной собственностью разработчиков и НЕ МОЖЕТ быть использован в коммерческих целях и/или для получения личной выгоды.
